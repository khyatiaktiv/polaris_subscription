Kanban Reordering

Before Scan Barcode on Screen We manage Kanban Re-ordering Rule for KRR and KFM

Configuration

Go to Settings - General Settings - Inventory and set :

- Multi-Warehouses

- Multi-Step Routes

- Storage Locations

- Internal Transfer Type for Kanban

Go to Settings - General Settings - Inventory - Multi-warehouses - Warehouses

Once you set Multi-Warehouses, Multi-Step Routes, Storage Locations,Internal Transfer Type for Kanban we need to manage Resupply Warehouses for that click on Warehouses from Settings and set True of Resupply Warehouses in Company Warehouses.

Go to Inventory - Dashboard - Receipt/Delivery Order - Settings

Once we set Resupply Warehouses we need to set Inter-Warehouse as True of selected company.

Once All Configuration is complete we can create re-ordering rule for KRR and KFM and scan Barcode in Kiosk Screen.

Go to Inventory - Kanban - Kanban for Supply Chain or Kanban for Manufacturing

Here we create Kanban for Supply Chain and one Auto Generated Barcode created and using this rule when user scan barcode they will get RFQ or Purchase Order Details on Scanner Screen.

Here we create Kanban for Manufacturing and one Auto Generated Barcode created and using this rule when user scan barcode they will get Kanban Reordering Rule Details for Physical type, Internal Transfer type, Inter-Warehouse type, Manufacturing Order type on Scanner Screen.
