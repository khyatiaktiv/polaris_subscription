Kanban Kiosk Barcode Scanner

Using barcode scanner user can get Manufacturing and Re-ordering details.

Step-By-Step Guidelines

Menu : Kanban --> Scan Kanban

Here is the screen called Kisok mode for barcode scanning and user can scan KFM and KRR barcode and get related details.

Once user click on Scan Barcode menu and open the screen user can enter barcode or user can scan by machine then user will get barcode related information when click on button.

Once User scan barcode they get Information related barcode, Here is the Re-ordering barcode so, user will get Kanban Reordering Rule Details and Purchase Details and click on confirm button last purchase order will update else create new one, user can also redirect to last RFQ when click on Hyperlink

Purchase Order:

If There is no related RFQ in system then it will create new one

If There is any related RFQ in system then it will Update last one

Here is the Manufacturing barcode and selected type is Physical so, user will get Kanban Reordering Rule Details.

Here is the Manufacturing barcode and selected type is Internal Transfer so, user will get Internal Transfer Details and Kanban Reordering Rule Details and when click on confirm button new internal transfer will create, user can also redirect to last Internal Transfer when click on Hyperlink

Internal Transfer:

Once User click on confirm button one new Internal transfer created and here user can see one message like create successfully.

Here is the Manufacturing barcode and selected type is Inter-Warehouse so, user will get Inter-Warehouse Details and Kanban Reordering Rule Details and when click on confirm button new internal transfer will create, user can also redirect to last Inter Warehouse(Delivery Order, Receipt) when click on Hyperlink

Delivery Order:

Receipt:

Once User click on confirm button new Delivery Order and Receipt created and here user can see one message like create successfully.

Here is the Manufacturing barcode and selected type is Manufacturing Order so, user will get Manufacturing Order Details and Kanban Reordering Rule Details and when click on confirm button new Manufacturing Order will create, user can also redirect to last Manufacture Order when click on Hyperlink

Manufacture Order:

Once User click on confirm button new Manufacturing Order created and here user can see one message like create successfully.
